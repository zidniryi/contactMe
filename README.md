# Contact Me

An App Contact API Android


# Usage
- npm i
- npm run android



# Technologies , Tools & About

- Redux
- React Native
- Axios
- Hooks
- Netinfo for check internet
- Pull To Refresh
- Test (enzyme, jest, enzyme-adapter-react-16, react-test-renderer)
